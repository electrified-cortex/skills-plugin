# markdown-hygiene - skill index

Covers the full hygiene pipeline for Markdown files, from automated lint fixes through semantic advisory analysis.

## markdown-hygiene

Runs a complete markdown hygiene pass — auto-fix, lint scan, and semantic advisory analysis — on a single .md file.

## markdown-hygiene-analysis

Performs semantic advisory scanning of a .md file against the SA001–SA038 rule set.

## markdown-hygiene-lint

Scans a .md file for MD rule violations, applying safe in-place fixes before the scan.

## markdown-hygiene-result

Probes the hash-record cache for the result of a markdown-hygiene sub-document (report, lint, or analysis).
